# bis-ai-recommendor
AI-powered system that recommends relevant BIS standards from product descriptions, helping MSEs quickly identify compliance requirements and reduce manual effort.
# 🚀 BIS AI Recommendation Engine

## 🧠 Problem
Micro and Small Enterprises (MSEs) ko relevant BIS standards identify karne me weeks lag jaate hain.

## 💡 Solution
Ek AI-powered recommendation system jo product description se instantly relevant BIS standards suggest karta hai.

## ⚙️ Approach
- Keyword-based retrieval system 🔍  
- Future-ready for RAG (Retrieval-Augmented Generation) pipeline 🤖  

## ✨ Features
- 📝 Input: Product description  
- 📊 Output: Top 3 relevant BIS standards  
- ⚡ Fast and lightweight system  

## 📌 Example
**Input:** Cement  

**Output:**
- IS 269  
- IS 8112  
- IS 456  

## 🛠️ Tech Stack
- 🐍 Python  

## 🔮 Future Scope
- 📦 Embeddings + FAISS integration  
- 🧠 LLM-based reasoning  
- 🌐 Streamlit UI  

## 🌍 Impact
⏱️ Compliance discovery time weeks se seconds me reduce karta hai for MSEs.
